# 🌌 AstraView（星辰视界）

> **告别一堆软件，从此一个窗口看尽万物。**

AstraView 不是看图软件，也不是播放器。  
它是一款**通用文件预览器**，让你在一个窗口中——  
🖼️ 浏览图片、🎬 播放视频、🎵 听音乐、📄 阅读 PDF、🧾 查看文档、🌐 打开网页。  

轻、快、干净、美观。  
不联网、不打扰、不乱写注册表。  
**一个窗口，取代你电脑上一半的工具。**

## ✨ 功能亮点

- 🖼️ **图片**：自动缩放，自适应窗口显示  
- 🎬 **视频**：原生播放，快速加载  
- 🎵 **音乐**：带动态波形背景的播放器  
- 📄 **PDF**：支持翻页、自适应、清晰渲染  
- 🧾 **文档**：支持 TXT / DOCX / XLSX / CSV / MD  
- 🌐 **网页**：支持本地 HTML 预览（内置 Chromium）  
- 🎨 **主题**：Light / Dark 一键切换  
- 🌍 **语言**：中 / 英 / 日 / 韩 / 西 / 德 / 葡 / 俄 / 阿
